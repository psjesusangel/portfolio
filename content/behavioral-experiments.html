As a Software Developer at Yale School of Medicine's Levy Neuro Decision Lab, I designed and built full-stack web applications for behavioral psychology research, enabling researchers to collect data on decision-making under risk and uncertainty.

Technical Architecture

  • Built modular, reusable JavaScript experiment frameworks using Node.js and Express
  • Implemented MongoDB for participant data storage and real-time experiment tracking
  • Created flexible configuration systems allowing researchers to modify experiments without touching code
  • Developed responsive interfaces ensuring consistent participant experience across devices

Key Projects

  • Risk Preference Studies: Interactive surveys measuring risk attitudes across different decision contexts, with randomized question ordering and adaptive feedback
  • Image Valuation Experiments: Custom interfaces for participants to rate and value images, with randomized slider positioning to reduce anchoring bias
  • Multi-stage Decision Tasks: Complex experiments involving sequential choices and probabilistic outcomes

Research Impact

  • Served 500+ participants across multiple studies
  • Provided researchers with clean, well-structured data for analysis
  • Enabled rapid iteration on experimental designs based on pilot results
  • Created reusable components that accelerated development of new experiments

Technical Challenges

  • Ensuring randomization and counterbalancing worked correctly across sessions
  • Building systems robust enough for participants with varying technical skills
  • Managing participant flow and data integrity across multi-session studies
  • Creating intuitive interfaces that didn't bias decision-making behavior
