Ari is a minimalist habit tracking application built as a native macOS app, designed around the philosophy that our habits define us more than our one-time achievements.

Design Philosophy

  • Infinite canvas interface for spatial organization of habits
  • Minimal UI that stays out of the way—focus on doing, not managing
  • No gamification, streaks, or artificial motivation—just honest tracking
  • Philosophy-driven: excellence through consistent action, not perfection

Technical Journey

  • Started as a web application using JavaScript and Canvas API
  • Evolved into a native macOS application for better performance and integration
  • Implemented smooth panning, zooming, and gesture controls for the infinite canvas
  • Built custom data persistence to maintain habit history locally

Key Features

  • Place habits anywhere on an infinite canvas
  • Simple check-in system without overwhelming statistics
  • Visual patterns emerge naturally from your tracking behavior
  • Offline-first with local data storage

The project represents a shift from typical productivity apps that rely on external motivation (points, streaks, social features) to one that simply provides a calm space for building sustainable habits.
