Conducted research on decentralized autonomous organization (DAO) governance structures, focusing on identifying DAOs that implement egalitarian (one-person-one-vote) systems rather than the typical plutocratic token-weighted voting.

Research Context

Most DAOs use token-weighted voting where voting power is proportional to token holdings, creating plutocratic governance structures. This research investigated whether genuinely egalitarian DAOs exist and how they function in practice.

Methodology

  • Used an internal dashboard tool to filter and analyze DAO governance data
  • Identified candidate DAOs by filtering for Gini coefficient = 0 (perfect equality in voting power distribution)
  • Collected a dataset of 60+ potentially egalitarian DAOs across platforms (Snapshot, DeepDAO, etc.)
  • Conducted qualitative analysis of each DAO's governance documentation, proposals, and voting records
  • Hand-labeled DAOs into categories based on their actual governance mechanisms

Analysis Framework

  • Egalitarian Governance Score (1-5): Based on actual voting mechanisms observed in practice
  • Self-Identification Score (1-5): Based on how DAOs describe their governance in documentation
  • Evaluated evidence from proposals, voting records, white papers, and community discussions
  • Documented contradictions between stated governance ideals and implemented systems

Key Findings

  • Very few DAOs truly implement one-person-one-vote despite rhetoric about decentralization
  • Many DAOs with egalitarian rhetoric still use token-weighted voting in practice
  • Some DAOs (like Fractaller) showed evidence of genuine egalitarian voting but lacked sustained activity
  • The gap between stated values and implemented systems reveals tensions in DAO governance design

Example Cases

  • Ether.fi: Token-weighted staking model (Score: 1) despite decentralization claims
  • Fractaller: One FRC per vote in proposals (Score: 4) but minimal documentation and single proposal
  • Study DAO: Appeared to be testing/learning project with plutocratic voting

The research highlighted the challenge of implementing truly egalitarian governance in crypto-native organizations where token economics often conflict with democratic ideals.
